Terrorist Targets is a brief exploratory analysis on the targets of some elected terrorist organizations. The dataset is taken from the Global Terrorism Database. A bar graph is created to look at which entities are the most popular targets that terrorists choose to attack. Then mosaic plots are created to show the differences in terrorist organizations, not only in who they attack, but also in how they carry out the attack. This raises important questions about terrorist organizations’ network structures, resources, and objectives, and how these variables interplay in terrorism in its entirety. If you’re interested in the code, feel free to email me; it’s currently hosted in a private repo.

Projecting with Python utilizes Python to clean a terrorism dataset and iterate through spatial data, transforming coordinates (which represent locations in 3D-space, like Google Earth) to projected values (basically coordinates for 2D-space, like most maps). The terrorist attacks are then mapped, retaining important attribute information, like the targets of the attack. The mapping is done in ArcGIS, and in the future I plan to map this data in Python.

Data Mining and Predictive Analytics solves R exercises in the book Data Mining and Predictive Analytics, using Python. The exercises involve data preparation, statistical analysis, classification, clustering, association rules, and model optimization.
