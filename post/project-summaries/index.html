Project Summaries

 Posted on January 3, 2017  |  2 minutes (311 words)

Here are some projects I’ve completed.

Data Pipeline for Species Distributions

for University researchers

The species distribution data pipeline and dashboard was developed to help scientists across the United States make data-driven decisions to protect wildlife from pesticides and controlled fires, with on-the-fly analyses and visualizations.

  • Led a team of developers in building a full data pipeline for analyzing species distributions
  • Developed a visualization dashboard for non-technical users with Python, Bash, and Git
  • Tested and prototyped with Python to ensure the product would scale reliably
  • Communicated with interdisciplinary teams, subject matter experts, and stakeholders in an Agile environment to ensure requirements were accurately analyzed and safely fulfilled

Authorship Attribution using Machine Learning

Being interested in privacy and pattern-finding, this project was undertaken for fun, for the sake of understanding how somebody online may be de-anonymized based on writing style. Stylometry has interesting applications in forensics, national security, and literature (think The Count of Monte Cristo).

  • Used Bash, Python, and an API to get the anonymous lyric data to be used in the analysis

  • Visualized and presented insights about the lyric classifications, like inferences about where the authors likely collaborated

Data Mining and Predictive Analytics

The varying Jupyter Notebooks involve data preparation, statistical analysis, classification, clustering, association rules, and model optimization.

  • Used Python for data mining and statistical analysis to find insights and anomalies in data
  • Implemented a k-nearest neighbors model in sklearn to classify individuals’ risk for credit loans
  • Developed a neural network to predict individuals’ income levels based on demographic features
  • Documented all code and explanations with Jupyter Notebooks for openness and reproducibility
